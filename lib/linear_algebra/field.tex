\begin{dfn}[体]	
	集合$F$が体とは，
	二項演算＋と×が定義されていて，次を満たすときをいう
	\begin{enumerate}
		\item $F$は和「＋」により可換群になっている．つまり，結合法則
			\[ (a+b)+c = a+ (b+c),\]
			単位元と呼ばれる元$0$があって，
			\[ a + 0 = 0 + a = a ,\]
			さらに，各$a\in F$に対して，逆元$(-a)\in F$があり，
			\[ a + (-a) = (-a) + a = 0,\]
			可換である．
			\[ a+b = b+a\]
		\item 積「$\times$」についても，「1」という元があり，
			\[ (a\times b)\times c = a\times (b\times c)\]
			\[ a\times b = b\times a\]
			\[ a\times 1 = 1\times a = a\]
			をみたし，
			$a\not = 0$のとき，ある元$a^{-1}$があって，
			\[ a \times a^{-1} = a^{-1} \times a = 1 \]
		\item 和と積について
			\[a\times (b+c) =a\times b +a\times c\]
		\item 非自明である．つまり，
			\[ 0\not= 1\]
	\end{enumerate}
\end{dfn}

混乱を招かない場合，積の記号を$\cdot$にしたり，$a\times b = ab$というように省略してもよい．

有理数全体$\mathbb{Q}$，実数全体$\mathbb{R}$，複素数全体$\mathbb{C}$が良くある例．

複雑な例
\[ \mathbb{Q}+\mathbb{Q}\sqrt{2} = \left\{\left.  x + y\sqrt{2} \; \right| x,y\in \mathbb{Q} \right\}\]
